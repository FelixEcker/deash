deash - purr




About purr


purr (Print User's Resource Record) is a built in deash command
to print out all non-sensitive data currently saved in the shell.
This command was originally used for debugging and the name was
chosen so that the cat-kaomoji could be used, thus the acronym is
abit non-sensical.

Output-Contents
1. Shell-Uptime
2. User-Data
3. Preffered Exp-Procs
4. Aliases
5. Binary Searchpaths
6. Exp-Procs
